Editor de Evidencias QA

Herramientas de Anotación

  • Atajos: Usa las teclas para activar el elemento de edición.
    • (F) Flecha: Señala elementos específicos.
    • (C) Cuadro: Enmarca errores o secciones.
    • (T) Texto Añade comentarios (Fuente ajustable)
    • (H) Resaltador Enfatiza áreas con transparencia.
    • (B) Blur Oculta datos sensibles o personales.
    • (R) Recortar Ajusta el tamaño de la evidencia.

    Personalización y Control

  • Color: Selecciona el tono de las herramientas (Rojo, Verde, Azul, etc.).
  • Grosor: Ajusta el ancho de las líneas y bordes.
    • Historial de Cambios:
      • Deshacer / Rehacer: Permite revertir o aplicar nuevamente las últimas acciones.

      Navegación y Zoom

      • Cancelar: Descarta los cambios y cierra el editor.
      • Guardar: Aplica todas las anotaciones y finaliza la evidencia.
      • Ctrl + Rueda : Aumentar o disminuir el zoom de la captura.
      Consejo de QA: Utiliza siempre la herramienta Blur para proteger información confidencial (nombres, cuentas, contraseñas) que aparezca en la captura.
