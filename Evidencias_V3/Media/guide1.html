Guía Rápida de Evidencias

✂️
Recorte de Precisión

Usa la herramienta Recortar o presiona R. Selecciona el área y pulsa Enter para confirmar el recorte.

🕵️
Anonimización (Blur)

Protege datos sensibles con la herramienta Blur. Dibuja sobre textos o rostros para pixelarlos automáticamente.

📋
Pega desde Portapapeles

Puedes presionar Ctrl + V en cualquier momento para adjuntar una imagen externa o externa a la evidencia actual.

⚡
Atajos Avanzados

Ctrl+S para guardar, Ctrl+D para duplicar objetos, y flechas de teclado para mover elementos con precisión.
